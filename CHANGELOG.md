# Changelog

All notable changes to the Nexus **Clears Tracker** addon are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added
- Initial Nexus port of the BlishHUD Clears Tracker module
- Raids, Strikes, Fractals, and Dungeons overlay panels
- GW2 API integration for weekly clears, daily raid bounties, strike achievement 9125, and mentor progress
- Options UI with per-content tabs, label customization, and encounter visibility
- Corner icon, keybind toggle, mentor progress popups, and fractal CM tooltips
- Identifiable `User-Agent` (`ClearsTracker-Nexus/1.0.0`) for requests to `bhm.blishhud.com` and `assets.gw2dat.com`
